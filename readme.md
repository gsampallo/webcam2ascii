# webcam2ascii

For fun, a simple python program to convert video from webcam to ascii art on the console.

## Installation

You need to install:

- OpenCV
  pip install opencv-contrib-python

## Usage

Clone the repository, then run:

```python
python w2i.py
```

You can exit the program pressing 'q' on the frame.
